# Data card

## Intended use

Datasets are used for educational and research experiments in agricultural decision-support modeling.

## Sources

- `data/raw/crop_recommendation.csv`
- `data/raw/fertilizer_dataset.csv`
- `data/raw/filtered_data.csv`

## Known caveats

- Limited representativeness for all agroecological regions.
- Possible class imbalance and noisy labels.
- Units and collection protocols may vary across sources.

## Recommended checks

- Validate ranges and missing values before training.
- Report split strategy and class distribution in results.
